# Shopping List Site

A simple shopping list web application built using HTML, CSS, and JavaScript. Users can type in items, add them to a list, and remove them with a click â€” all without page reloads.

## Features

- Add items to your shopping list
- Remove individual items with a Delete button

- Ignores empty or blank entries
